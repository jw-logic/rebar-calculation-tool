# Rebar Calculation Tool
## Overview
The tool includes a **rebar area lookup table** showing the cross-sectional area for various bar diameters and quantities. 
Once you enter the **required area** from your analysis, the tool calculates a **rebar count guide** based on single bar diameters.

Using this as a reference, you can input your final **rebar combination**. The tool then calculates the **percentage difference** 
between the provided and required area.

By entering the **sectional dimensions** of the concrete, you can determine the **percentage of rebar provided** to 
ensure it meets code requirements. The tool also provides an option to view **tension steel limits** based on your 
specific concrete and steel grades.

## Key Features
* **Rebar Count Guide:** Calculates the number of rebars needed for a single diameter.
* **Combination Input:** Allows for a mix of different bar diameters.
* **Area Verification:** Shows the percentage exceeding or falling short of the target $A_s$.
* **Reinforcement Ratio:** Calculates the actual percentage of steel provided relative to the concrete section.
* **Steel Limits:** Displays tension steel limits based on the selected concrete and steel grades.

## Important: Enable Macros ⚠️
Because this file contains VBA code, Windows will block it by default for your safety. To use it:
1. **Right-click** the downloaded file in your folder.
2. Select **Properties**.
3. At the bottom of the "General" tab, check the box that says **Unblock**.
4. Click **Apply/OK**, then open the file and "Enable Content."
